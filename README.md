# Frozen 2 English Explorer ❄️

A simple web application designed to teach a 4-year-old English through "Wh-" questions (Who, What, Where, When, Why) and spatial words (Here, There).

## How to use
1. Clone or download this repository.
2. Open `index.html` in any web browser.
3. Click the buttons to explore 50 different Frozen 2 scenarios!
